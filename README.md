# JOGO BOT 🔥

A powerful **Jujutsu Kaisen themed WhatsApp bot** inspired by the Special Grade Curse **Jogo**.  
Built for speed, automation, intelligence, and powerful group management.

JogoBot combines **AI features, educational tools, media utilities, and fun commands** to create a complete WhatsApp bot experience.

• Mathematics  
• English  
• Physics  
• Chemistry  
• Biology  
• Economics  
• Government  
• Literature  
• Geography  
• Commerce  
• Accounting
