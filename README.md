# JOGO BOT 🔥

A powerful Jujutsu Kaisen themed WhatsApp bot inspired by the Special Grade Curse, **Jogo**. Built for speed, efficiency, and a touch of volcanic arrogance.

## 🌋 Features

- **Core**: Alive, Ping, Menu, Owner, Restart
- **Admin**: Kick, Add, Promote, Demote, Tagall, Mute/Unmute
- **Auto**: Auto-Sticker, Auto-Read, Anti-Link, Anti-Delete
- **Media**: Sticker, Video/Audio Downloader, Image Search
- **Fun**: Roasts, Truth/Dare, Anime Quotes

## 🚀 Installation

### Replit (Easiest)
1. Fork this repository.
2. Click the **Run** button.
3. Scan the QR code in the console.

### VPS / Local
```bash
git clone https://github.com/OAD-26/JOGO-BOT.git
cd JOGO-BOT
npm install
node index.js
```

## ⚙️ Configuration
Edit `config.js` to set your owner number and customize bot behavior.

| Variable | Description |
|----------|-------------|
| `SESSION_ID` | Pre-authenticated session string |
| `ownerNumber` | Your WhatsApp number (91...) |
| `prefix` | Bot command prefix (default: `.`) |
